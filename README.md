# Openclassroom Developpeur web P6 Piiquante

__Prérequis :__  
* Angular CLI
* Node.js
* mongoose

__Instruction :__  
  
1. Clonez ce repository  

__Frontend :__  
  
1. Suivez les instruction du *README.md* dans le dossier *Frontend*  
2. Exécuter  la commande suivante
> npm run start  

__Backend :__  
  
1. Exécuter la commande suivante
> nodemon server
